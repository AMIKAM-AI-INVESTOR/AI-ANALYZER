
# 📊 AI Crypto & Stock Analyzer

מערכת ניתוח חכמה למניות ומטבעות קריפטוגרפיים, המבוססת על בינה מלאכותית.

## תכונות עיקריות:
- ניתוח גרפי: גרף נרות יפניים עם איתותי קנייה/מכירה
- תחזיות AI עם הסבר וטיעון
- תבניות גרפיות (ראש וכתפיים, דגל שורי)
- פילטרים לפי סיכון, תחזית, סוג נכס
- מצב כהה ועיצוב מקצועי
- מותאם למובייל/טאבלט

## הוראות שימוש:
1. העלה את הקבצים ל-Streamlit Cloud (כולל model_engine.py, ai_crypto_analyzer.py, pattern_detection.py)
2. ודא שיש לך קבצי עזר utils.py, backtesting.py, fundamentals.py
3. לחץ Run והמערכת תיטען
4. השתמש בפילטרים בצד שמאל לסינון תחזיות
5. עבור לגרף ניתוח לפי סימול להזין מניות או מטבעות שתרצה לבדוק

## דרישות:
- Python 3.8+
- ספריות: streamlit, xgboost, yfinance, pandas, plotly, joblib

בהצלחה! 🎯
